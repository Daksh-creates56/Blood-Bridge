# **App Name**: Blood Bridge

## Core Features:

- Real-time Inventory Dashboard: Display real-time blood inventory levels with filtering and search capabilities for blood banks and hospitals.
- Urgent Request Broadcast: Enable hospitals to send urgent blood requests with specified blood type, quantity, urgency level, and broadcast radius.
- Donor-Hospital Matching: Alert donors and blood banks to urgent blood requests and facilitate connection for donations.
- Donation Camp Management: Allow organizers to create, display, and manage blood donation camps with integrated registration.
- Interactive Camp Locator Map: Display an interactive map with the locations of donation camps.
- Data Visualization: Show a dashboard with a bar chart to display blood inventory by blood type and status, to give medical professionals information.
- Smart Alerting: Use an AI tool to identify critical shortage patterns and predict urgent needs, prioritizing alerts to relevant stakeholders for pre-emptive action.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Light blue (#E8EAF6), subtly desaturated, to maintain a clean and professional feel.
- Accent color: Complementary orange (#FF9800) for interactive elements and urgent notifications.
- Body and headline font: 'Inter', a sans-serif, for a modern, clean, and neutral reading experience.
- Use consistent Lucide React icons with a clean and modern style to represent different functionalities.
- Implement a collapsible sidebar layout for easy navigation and focus on main content areas.
- Employ subtle animations for splash screen, transitions, and alerts to improve user experience without distraction.
